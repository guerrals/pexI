# Paleta de Cores Aplicada

Baseada na análise da imagem fornecida, a seguinte paleta de cores foi implementada no tema Material-UI:

## Cores Principais

### Vermelho Principal
- **Hex:** `#B71C1C` (RGB: 183, 28, 28)
- **Uso:** Cor primária para botões, abas ativas, títulos principais
- **MUI:** `palette.primary.main`

### Dourado
- **Hex:** `#FFD700` (RGB: 255, 215, 0)
- **Uso:** Cor secundária para contraste, texto em botões primários
- **MUI:** `palette.secondary.main`

### Vermelho Vibrante
- **Hex:** `#FF0000` (RGB: 255, 0, 0)
- **Uso:** Destaques importantes, alertas de erro
- **MUI:** `palette.error.main`

### Azul para Links
- **Hex:** `#0000EE` (RGB: 0, 0, 238)
- **Uso:** Links clicáveis
- **MUI:** `palette.info.main`

## Cores de Fundo

### Cinza Claro
- **Hex:** `#F5F5F5` (RGB: 245, 245, 245)
- **Uso:** Fundo padrão da aplicação
- **MUI:** `palette.background.default`

### Branco
- **Hex:** `#FFFFFF` (RGB: 255, 255, 255)
- **Uso:** Fundo de papel, modais, cards
- **MUI:** `palette.background.paper`

## Cores de Texto

### Cinza Escuro
- **Hex:** `#212121` (RGB: 33, 33, 33)
- **Uso:** Texto principal, cabeçalhos
- **MUI:** `palette.text.primary`

### Cinza Médio
- **Hex:** `#757575` (RGB: 117, 117, 117)
- **Uso:** Texto secundário
- **MUI:** `palette.text.secondary`

## Componentes Customizados

### Botões
- **Primário:** Fundo vermelho (`#B71C1C`) com texto dourado (`#FFD700`)
- **Secundário:** Fundo dourado (`#FFD700`) com texto vermelho (`#B71C1C`)

### Abas
- **Indicador:** Vermelho principal (`#B71C1C`)
- **Aba Ativa:** Texto vermelho (`#B71C1C`)

### Campos de Formulário
- **Foco:** Borda vermelha (`#B71C1C`)
- **Label Focado:** Texto vermelho (`#B71C1C`)

### Botão Flutuante (FAB)
- **Fundo:** Vermelho principal (`#B71C1C`)
- **Ícone:** Dourado (`#FFD700`)

Esta paleta cria uma identidade visual forte e profissional, mantendo boa legibilidade e acessibilidade.
